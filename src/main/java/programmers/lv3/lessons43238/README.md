### 문제 풀이 접근법

- 사람 베이스로 풀 것인가?
  - 6명의 사람이 있고, 7분, 10분에 끼워보자
  - 사람을 대상으로 문제 풀이를 하면 시간초과가 난다.
- 시간 테이블을 기준으로 풀어볼까?
  - 가장 오래 걸릴 수 있는 시간대를 직접 설정한다. 이분 탐색의 end node
  - start node: 가장 빨리 끝날 수 있는 시간대
  - start와 end 사이에서 이분 탐색을 진행한다.
